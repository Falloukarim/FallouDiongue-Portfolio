Système de Contrôle d'Accès RFID Intelligent

Application IoT complète pour la gestion d'accès sécurisée

Système de contrôle d'accès en temps réel utilisant la technologie RFID avec ESP32. Les cartes sont authentifiées via une base de données centralisée et les événements sont visualisés sur une interface web en temps réel.

Fonctionnalités :

  • Authentification RFID en temps réel
  • Interface web avec React et Socket.IO
  • Base de données MySQL avec synchronisation
  • Historique complet des accès
  • Système de notifications en temps réel
  • API REST pour gestion des utilisateurs

Technologies : ESP32 (C++/Arduino), React.js, Node.js, Socket.IO, MySQL, Railway (déploiement)
